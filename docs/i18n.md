# Đa ngôn ngữ (i18n)

Dự án hỗ trợ **tiếng Anh (`en`)** và **tiếng Việt (`vi`)** qua
[`react-i18next`](https://react.i18next.com/).

## Cấu hình — `src/Assets/lang/i18n.js`

Toàn bộ chuỗi dịch nằm trong **một object `resources` duy nhất** với hai khối
`en.translation` và `vi.translation`. Đây là single source of truth — không có
file JSON tách riêng.

Điểm cấu hình quan trọng:

```js
i18n.use(initReactI18next).init({
  resources,
  lng: localStorage.getItem('language') || 'en',  // ngôn ngữ đã lưu, mặc định en
  fallbackLng: 'en',
  interpolation: { escapeValue: false },
  react: {
    useSuspense: false,
    transSupportBasicHtmlNodes: true,
    transKeepBasicHtmlNodesFor: ['br', 'strong', 'i', 'p'],
    // ...
  }
});
```

- Ngôn ngữ được **ghi nhớ trong `localStorage['language']`** bởi `LanguagePicker`.
- `escapeValue: false` cho phép một số chuỗi chứa HTML.

## Cách dùng trong component

```jsx
import { useTranslation } from "react-i18next";

function MyComponent() {
  const { t, i18n } = useTranslation();
  return <h1>{t('Home')}</h1>;            // tra key 'Home'
}
```

Đổi ngôn ngữ:

```js
i18n.changeLanguage('vi');
localStorage.setItem('language', 'vi');
```

(Logic này đã có sẵn trong `LanguagePicker`.)

## Thêm / sửa chuỗi dịch

1. Mở `src/Assets/lang/i18n.js`.
2. **Thêm key vào CẢ HAI khối** `en.translation` và `vi.translation`. Thiếu một
   bên sẽ fallback về `en` (hoặc hiện key thô nếu cả hai thiếu).
3. Dùng `t('Key của bạn')` trong JSX.

Quy ước key trong dự án: dùng chính câu tiếng Anh làm key (vd `'Home'`,
`'My Recent'`), hoặc key mô tả cho đoạn dài (vd `'Food Ordering Description'`).

## Chuỗi chứa HTML

Một số key — đặc biệt các `*Description` của project — chứa thẻ HTML như
`<br/>`, `<b>`. Chúng được render bằng `dangerouslySetInnerHTML` trong
`ProjectCards.js`:

```jsx
<Card.Text dangerouslySetInnerHTML={{ __html: props.description }} />
```

Khi sửa các key này, giữ nguyên các thẻ HTML hợp lệ. Với text nhúng trong cây
React (không qua `dangerouslySetInnerHTML`), `react-i18next` chỉ giữ các thẻ liệt
kê trong `transKeepBasicHtmlNodesFor` (`br`, `strong`, `i`, `p`).

## Lưu ý

- Đừng hardcode chuỗi tiếng Anh/Việt trực tiếp trong JSX — luôn đi qua `t()` để
  giữ tính song ngữ.
- Một vài label tĩnh (vd "Codeforces", "Download CV", "Deployed", địa chỉ email,
  SĐT) hiện đang hardcode, không qua i18n — nếu cần dịch thì phải thêm key mới.
